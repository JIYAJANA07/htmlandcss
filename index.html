CSS Box Model & Flexbox — Code + Answers

Task 1

Padding increases inner space

    ANSWERS:
  1. - Padding increases visible size of element
  2. - Padding does NOT directly affect nearby elements
  3. - Padding belongs to the element itself

Task 2

Margin creates outer space

    ANSWERS
  1. - Margin does NOT increase element size
  2. - Margin does NOT belong to the element
  3. - Margin creates space outside the border

Task 3

Padding colored
Margin invisible

    ANSWERS:
  1. - Padding area is colored
  2. - Margin is transparent
  3. - Background applies to content + padding only

  3. Shorthand improves readability when values are logical
  4. Individual properties improve clarity when values differ

Task 5

Negative margin works

    ANSWERS
  1. - Negative padding is invalid
  2. - Negative margin is allowed
  3. - Negative margins pull elements closer or overlap

Task 20 – FINAL RULES

    RULES FOR MAINTAINABLE CSS
  1. Padding is for internal spacing only
  2. Margin is for separating components
  3. Never use margin for alignment
  4. Use flexbox for layout & centering
  5. Let containers control spacing, not children
